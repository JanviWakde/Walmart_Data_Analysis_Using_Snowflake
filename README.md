# Walmart_Data_Analysis_Using_Snowflake

## Features:
- Loaded CSV sales data into Snowflake.
- Data cleaning and validation.
- Created star schema (fact & dimension tables).
- Business analysis queries: total sales, sales by branch, top products, customer segmentation, monthly sales.
- Views for reporting.
- Advanced metrics: average rating, average order value per branch.
- Optimized for performance and security.

## Tools:
- Snowflake (Cloud Data Warehouse)
- SQL
